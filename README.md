# 11-frontend

## Task: Hover Image Hide/Show

This folder contains a simple frontend code where an image disappears when the mouse hovers over it and reappears when the mouse moves away.

### Files Included
- `index.html` → JavaScript version (hide/show on hover)
- `index-css.html` → CSS-only version (hide/show on hover)

### How to Run
Open the file in any browser:

- Double click the file OR
- Run in terminal (Kali Linux):
  ```bash
  firefox index.html
